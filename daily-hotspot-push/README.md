# 每日行业热点定时推送服务
每日早9点自动采集AI、互联网、科技三个领域24小时内热点，推送到指定飞书群，支持手动补发、失败重试。

## 功能特性
✅ 定时触发：每日早9点自动执行，时间误差≤5分钟
✅ 多领域采集：AI、互联网、科技三个领域，每个领域3-5条热点
✅ 结构化输出：每条包含领域标签、标题、100字内摘要、来源链接
✅ 异常处理：无热点自动提示、推送失败重试2次、支持手动补发
✅ 飞书原生样式：严格按照设计规范输出，全端兼容

## 配置步骤
1. 复制环境变量示例文件：`cp .env.example .env`
2. 配置飞书机器人Webhook：在`.env`中填写`FEISHU_WEBHOOK`（飞书群设置->机器人->添加自定义机器人获取，安全设置勾选"签名校验"或"IP白名单"）
3. （可选）调整推送时间：修改`PUSH_TIME`参数，格式为`HH:MM`
4. 安装依赖：`pip install -r requirements.txt`

## 部署方式
### 1. 后台持久化运行
```bash
nohup python main.py > run.log 2>&1 &
```

### 2. 查看运行状态
```bash
ps aux | grep main.py | grep -v grep
```

### 3. 手动触发补发
```bash
python main.py manual
```

## 输出样式说明
严格遵循设计规范：
- 头部：📰 每日行业热点 | YYYY-MM-DD + 灰色统计说明 + 分割线
- 领域标签：AI(#165DFF蓝色)、互联网(#00B42A绿色)、科技(#FF7D00橙色)
- 热点条目：序号 + 加粗带链接标题 + 灰色缩进100字内摘要
- 底部：灰色说明文字
